# Prompt Clarity Website

Marketing website for [Prompt Clarity](https://github.com/lucidgeo/lucidgeo) - Track and analyze your brand's visibility across AI-powered search platforms.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
